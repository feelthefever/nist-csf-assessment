# NIST CSF Security Maturity Assessment & Remediation Roadmap

## Project Overview
This repository contains a comprehensive security maturity assessment aligned with the **NIST Cybersecurity Framework (NIST CSF)**. I evaluated the organization’s current security posture across the five core functions: **Identify, Protect, Detect, Respond, and Recover**, and produced a **3-year remediation roadmap** based on the identified cybersecurity gaps.

## Scenario
A mid-sized organization wants to design a comprehensive security program. To do this, they engaged an external auditor who is a GRC analyst to highlight their current deficiencies using the NIST CSF framework and then include a 3-year remediation roadmap to improve their cybersecurity posture based on the deficiences found. GRC analysts in the company have given the external auditor a list of aspects of the company's current cybersecurity posture.

## Artifacts
This project has two documents: the spreadsheet used to determine if the organization passes or fails each control in the NIST CSF framework and the pdf document containing the organization's 3-year remediation roadmap, containing steps the organization should take over the next 36 months to significantly improve their cybersecurity posture.
